# 琴乐启蒙 AI 导师 · 乐谱驱动测试版

这是一个新的网页原型，目的是验证“用乐谱文件替代人工标准音频”的路线。

## 功能

1. 上传 MusicXML，自动生成曲目画像：调性、节拍、音符数、音区、核心音、旋律轮廓、教学重点。
2. 保存曲目卡到浏览器本地存储。
3. 学生选择曲目后上传演奏音频。
4. 网页做基础音频质量检测、有效片段提取、粗略音高统计、曲目匹配判断。
5. 曲目不匹配时不评分；通过后生成五维测评、学生版反馈、老师版反馈、Tomplay 练习建议。

## 启动

```bash
npm install
npm start
```

打开：

```text
http://localhost:3000
```

如果需要用 Cloudflare Tunnel：

```bash
cloudflared tunnel --url http://localhost:3000
```

## 注意

这是课堂展示与算法路线验证原型。浏览器端音高检测对复调钢琴音频仍是粗略分析，不能替代专业逐音识别。正式版建议接入后端音频分析模块或 MIDI/MusicXML 对齐算法。
